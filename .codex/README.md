# Repo-Local Codex Setup

This folder is for Codex context that should travel with this Rust lessons
repository.

## Layout

- `skills/` - repo-specific skills. Add each skill in its own folder with a
  `SKILL.md` entrypoint.

There are no repo-specific skills yet.

## Skill Shape

Use this structure for future custom skills:

```text
.codex/
  skills/
    your-skill-name/
      SKILL.md
      scripts/
      examples/
      templates/
```

Only `SKILL.md` is required. Add `scripts/`, `examples/`, or `templates/` when
the skill needs reusable assets.
